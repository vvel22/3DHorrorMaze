# 3DHorrorMaze
The player must go through a maze with certain objectives while there is a monster AI that chases them.

Title: Lost in the Fog

Members: Only me so far unless someone joins or a group adopts me, Vince Velasco

Main Idea: A slight tweak on Slenderman, a forest background and you have to find these certain keys in order to open this gate on the other side of the forest. Og course these keys will be scattered all throughout the forest and the catch is you are not alone. There will be an AI driven monster that patrols around the forest and will run after you if you are spotted. There will be a lot of details that need to go into sound design and the animation here in order to make this feel real scary. A timer will be added for speedrunning.

General Goals:
Animation: Try to create at least 3 varying jumpscare screens, have a good flashlight animation, and have good movement animation for monster
Audio: Create a captivating sound design that really encapsulates the fear that the game is supposed to bring.
Code: The monster AI will most likely be the hardest thing to implement so that will be prioritized first. The key system and the player movement should be pretty straight forward.
